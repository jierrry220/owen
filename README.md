# 🔐 Debear Party - Owner Control Panel

**私密管理控制面板** - 仅供项目管理员使用

⚠️ **安全警告**: 此控制面板包含敏感管理功能,必须部署在受保护的服务器上,不应公开访问!

## 🎯 功能

- 📊 **数据统计** - 查看平台数据和用户统计
- 👥 **用户管理** - 管理用户账户和权限  
- 💰 **余额管理** - 监控和调整用户余额
- 📈 **交易监控** - 实时监控平台交易
- ⚙️ **系统配置** - 修改系统参数和设置

## 🔒 安全建议

### 必须执行:
1. ✅ **独立部署** - 与前端分开部署
2. ✅ **密码保护** - 设置强密码
3. ✅ **IP 白名单** - 只允许特定 IP 访问
4. ✅ **HTTPS** - 必须使用 SSL 加密
5. ✅ **二次认证** - 建议添加 2FA

### 推荐措施:
- 🔐 使用 VPN 访问
- 🚫 不要分享访问链接
- 📝 定期检查访问日志
- 🔄 定期更换密码
- 🛡️ 使用防火墙规则

## 🚀 部署选项

### 选项 1: 私有服务器(推荐)
部署在只有你能访问的服务器上

### 选项 2: 本地运行(最安全)
```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080
```

## 📂 文件说明

- `index.html` / `admin.html` - 主控制面板
- `stats.html` - 数据统计面板
